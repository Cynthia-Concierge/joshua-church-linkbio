# Joshua Dean Church - Link in Bio

Simple, mobile-first link-in-bio page for Instagram.

## Auto-Deploy

This repo is linked to Vercel. Every push to `main` triggers an automatic deployment.

**Live URL:** Will be available at `joshua-church-linkbio.vercel.app` once Vercel detects the repo.

## How to Update

1. Edit `index.html`
2. `git add . && git commit -m "Update message"`
3. Push happens automatically via git hook
4. Vercel deploys automatically

No manual deploy commands needed.
